Welocome to our Project

This project primarily used the embedded systems microcontroller MSP432P401R.

The project comprised of using a master MSP and slave MSP via connected through bluetooth, one giving the instructions for direction and the slave sending the receiving data to the speed controller and outputting the chirp to the two motors. There was a source of power using 12 Volts.
